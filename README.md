BaseCoords by Amina El Houari

BaseCoords qu'est ce que c'est?
Petit plugin pour save ses bases sans avoir à sortir un papier ou screenshot des coordonnées toutes les 2 min. Tout est stocké proprement sur le serveur.

Commandes 
/setbase <nom> : Tu crées et enregistre les co où tu es actuellement 

/base <nom> : Ça t'affiche les coo dans le chat.
